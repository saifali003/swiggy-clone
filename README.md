A modern Swiggy-inspired food delivery web application built using React.js & Redux-Toolkit  
This project fetches live restaurant and menu data using Swiggy APIs and provides a smooth food ordering experience.

---

## 🚀 Features

- 🍕 Browse restaurants
- 🔍 Search functionality
- 📋 Dynamic restaurant menus
- 🛒 Add to cart functionality
- ⭐ Ratings and pricing display
- 🥗 Veg / Non-Veg indicators
- 📱 Responsive UI
- ⚡ Fast performance with React

---

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- Swiggy Live API
- Vite
